FermentStation

Fermented beverages such as kombucha have become popular in the past few years as a delicious, healthy alternative to soft drinks. I recently took up the hobby of making them at home. It’s fun and a great way to save money, but it also requires a lot of experimentation. I built an app that monitors the fermentation stages of each batch and allows me to record and compare ingredients. FermentStation is a single-page CRUD app built in React with a flat data structure in JSON server.

GoalFish

My friend is a special education teacher, and tracking a student’s behavior goals is a key part of her job. This data helps her to plan lessons and communicate with parents about their child’s progress. Currently she records scores on paper and uses a calculator for averaging. I built an app that allows her to input scores, calculate averages automatically, and output the student's scores in a table. This saves her hours of time and allows her to easily visualize a student's progress. GoalFish is a server-generated web app built using Python with Django web framework and styled with Bootstrap.
